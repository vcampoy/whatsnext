# whatsnext
A todo list

Usage

- **Install:** `yarn` or `yarn install`
- **Dev server:** `yarn dev`
- **Build:** `yarn build`
- **Preview build:** `yarn preview`
- **Tests:** `yarn test` (UI: `yarn test:ui`, run: `yarn test:run`, coverage: `yarn test:coverage`)

Notes

- This project uses Yarn and includes a `yarn.lock` file. Run `yarn` to generate or update the lockfile.
